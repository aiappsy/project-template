Pips in Forex Trading

A Pip (Percentage in Point) is the standard unit of measurement in forex trading, representing the smallest price move that a currency pair can make.

How Pips Work

For most currency pairs, a pip is 0.0001 (1/100th of 1%) of the exchange rate.

Example: EUR/USD

Price moves from 1.1050 to 1.1051 = 1 pip movement

Price moves from 1.1050 to 1.1060 = 10 pip movement

For JPY pairs, a pip is 0.01 (1% of 1 yen):

USD/JPY moves from 110.50 to 110.51 = 1 pip movement

Pip Values by Lot Size

Standard Lot (100,000 units):
Pip Value = (0.0001 / Exchange Rate) √ó 100,000
For EUR/USD at 1.1050: (0.0001 / 1.1050) √ó 100,000 = $9.05 per pip

Mini Lot (10,000 units):
Pip Value = $9.05 √ó 0.1 = $0.91 per pip

Micro Lot (1,000 units):
Pip Value = $9.05 √ó 0.01 = $0.09 per pip

Fractional Pips (Pipettes)

Many brokers now quote to 5 decimal places for most pairs and 3 decimal places for JPY pairs. The fifth decimal place (or third for JPY) is called a pipette, which is 1/10th of a pip.

EUR/USD moving from 1.10500 to 1.10501 = 1 pipette movement

Practical Pip Calculation

EUR/USD Position
Entry: 1.1050
Current: 1.1075
Movement: 25 pips
Position Size: 2 standard lots
Pip Value: $9.05 √ó 2 = $18.10
Profit: 25 pips √ó $18.10 = $452.50

Important Note on JPY Pairs

For JPY pairs, the pip value calculation is different because the pip is 0.01 instead of 0.0001:

USD/JPY Pip Value = (0.01 / Exchange Rate) √ó Position Size

At USD/JPY 110.00: (0.01 / 110.00) √ó 100,000 = $9.09 per pip

Ticks in Futures

In futures trading, a Tick is the minimum price movement of a contract. Unlike pips and points, tick sizes and values are fixed for each futures contract.

How Ticks Work

Each futures contract has:

Tick Size: Minimum price fluctuation

Tick Value: Dollar value of one tick

Example: E-mini S&P 500 (ES)

Tick Size: 0.25 points

Tick Value: $12.50 per contract

Price moves from 4500.00 to 4500.25 = 1 tick movement = $12.50 profit

Satoshis in Crypto

In cryptocurrency trading, a Satoshi is the smallest unit of Bitcoin, named after its creator Satoshi Nakamoto. One Bitcoin equals 100 million Satoshis.

Understanding Satoshis

1 Bitcoin (BTC) = 100,000,000 Satoshis

1 Satoshi = 0.00000001 BTC

This allows for precise measurement of small price movements in Bitcoin and other cryptocurrencies.

Satoshis in Practice

Bitcoin Price Movement

To: $40,001.00

Why Satoshis Matter

As Bitcoin's price increases, dollar-based measurements become less precise for small movements. Satoshis provide a consistent measurement unit regardless of Bitcoin's dollar value.

Professional crypto traders often think in Satoshis rather than dollars for precise position sizing.

Position Sizing Formulas

Now that we understand the different measurement units, let's explore how to use them for precise position sizing across all asset classes.

The Universal Position Sizing Formula

Regardless of asset class, the core position sizing formula remains the same:

Position Size = (Account Risk / Trade Risk) √ó Unit Value

Where:

Account Risk: Amount you're willing to lose (e.g., 1% of account)

Trade Risk: Distance to stop loss in asset's units

Unit Value: Value of one unit movement

Forex Position Sizing

Account: $10,000
Risk per trade: 1% = $100
Trade: EUR/USD
Entry: 1.1050
Stop Loss: 1.1020 (30 pips risk)
Pip Value for 1 lot: $10

Position Size = ($100 / 30 pips) √ó (1 lot / $10 per pip)
= 3.33 √ó 0.1 = 0.33 lots

Stock CFD Position Sizing

Account: $10,000
Risk per trade: 1% = $100
Trade: Apple (AAPL)
Entry: $150.00
Stop Loss: $148.50 (1.5 points risk)
Share Value: $1 per point per share

Position Size = ($100 / 1.5 points) √ó (1 share / $1 per point)
= 66.67 √ó 1 = 67 shares

Index CFD Position Sizing

Account: $10,000
Risk per trade: 1% = $100
Trade: S&P 500 (US500)
Entry: 4500
Stop Loss: 4480 (20 points risk)
Point Value: $50 per point per contract

Position Size = ($100 / 20 points) √ó (1 contract / $50 per point)
= 5 √ó 0.02 = 0.1 contracts

Position Sizing Principles

  • Consistency: Always use the same risk percentage
  • Precision: Calculate exact position sizes
  • Adaptability: Adjust for different asset classes
  • Risk Management: Position size determines your maximum loss
